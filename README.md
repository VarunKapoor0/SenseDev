# SenseDev

SenseDev is a desktop tool for **visual exploration and understanding of software codebases**.

It is designed to help developers quickly build a mental model of how a codebase is structured, how components relate to each other, and where important logic lives â€” especially when working with unfamiliar or large projects.

This repository contains the **developer preview** of SenseDev.

---

## What SenseDev Is

- A **standalone desktop application**
- Focused on **codebase structure and relationships**, not editing
- Built for **developers onboarding onto new or complex codebases**
- A visual, graph-based aid for reasoning about code

---

## What SenseDev Is NOT

- Not an IDE
- Not a code editor
- Not a refactoring or code-generation tool
- Not a replacement for VS Code or JetBrains IDEs
- Not a runtime debugger

SenseDev is intentionally **read-only** and **analysis-focused**.

---

**Version:** v0.1.0 (Developer Preview)

---

## Supported Platforms

- **Windows (x64)**

macOS and Linux are not supported yet.

---

## Installation (Windows)

1. Download the installer from the GitHub Releases page:
   - `SenseDev-0.1.0.exe`
2. Run the installer and follow the on-screen instructions
3. Launch SenseDev from the Start Menu after installation

No additional runtime or Java installation is required.

---

## Basic Usage

1. Launch SenseDev
2. Open a project directory containing a codebase
3. Explore the generated structure and relationships visually
4. Use the interface to inspect components and their connections

SenseDev does not modify your files.

---

## Known Limitations

- This is an early preview; stability may vary
- Performance may degrade on very large codebases
- No auto-update mechanism
- Windows-only at this stage

Any limitations discovered will be tracked via GitHub Issues.
